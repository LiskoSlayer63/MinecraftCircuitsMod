AnalogToDigital():

This circuit takes a redstone signal as input and outputs the power level (0-15) of the input signal in binary on its 4-bit output. 

Examples:
Full-strength redstone signal -> 1111 on output

Strength 7 redstone signal -> 0111 on output

No redstone signal -> 0000 on output

See DigitalToAnalog() for a circuit which does the opposite
